# Django图书馆可视化管理系统 python977

## 项目概述

本项目是基于Python的Django框架开发的图书馆可视化管理系统，旨在提供一套功能全面、便于管理的图书馆解决方案。

## 技术栈

- 后端：Python + Django
- 数据库：MYSQL
- 前端：B/S架构
